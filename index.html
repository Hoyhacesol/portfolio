브랜드 홍보 페이지

브랜드 홍보 페이지

미국의 자동차 회사 Tesla의 홍보 페이지를 제작해 보았습니다.
html으로 기본적인 구조를 잡은 후 배경화면으로 테슬라의 자동차 Model 3의 영상을 넣고, skewX속성을 사용해 일반적 직선 형태가 아닌 대각 형태의 네비게이션 바를 사용했습니다.
중앙에는 드롭다운 목록을 사용해 자동차의 옵션을 고를 수 있게 하였고, 오른쪽 아래에는 무한 애니메이션을 통해 방문자의 커서를 유도하는 효과를 넣었습니다.
사용자의 커서가 닿게 되면 추가적인 컨텐츠 메뉴들이 열리게 되며, 해당 메뉴에는 동영상·사진을 통한 링크나 뉴스 내용이 담긴 Carousel을 사용했습니다.
